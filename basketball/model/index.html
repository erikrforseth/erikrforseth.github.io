About the model

My model is a linear regression formula trained on 25,000+ regular season games from 2010-2016, which takes as input a number of season-to-date team statistics, some of which are raw (e.g. average margin of victory) and some of which are adjusted according to teams' opponents' statistics. For these adjusted measures, I presently use something of a generalization of Andrew Dolphin's "infinite" RPI routine, described here and here. For most of the analysis that goes into this, I use pandas, statsmodels, and scikit-learn, though I wrote my own routines for e.g. feature selection.
